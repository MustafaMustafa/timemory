Detailed Description

A decorator for the auto-timer, e.g.:
    @timemory.util.auto_timer(add_args=True)
    def main(n=5):
        for i in range(2):
            fibonacci(n * (i+1))
    # ...
    # output :
    # > [pyc] main(5)@'example.py':10 ...
